This API supports 3 requests: 1) A GET request for creating a new connection. This can be done using the url below ---
localhost:3000/api/request?connId=3&timeout=30000
2) A GET request to get the details for all connections running on the server. This can be done using the url below.
-localhost:3000/api/serverStatus
3) A PUT request to kill a particular connection id. This can be done using the url below--
-localhost:3000/api/kill/3(or connection id
